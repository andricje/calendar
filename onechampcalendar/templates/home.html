ONE Championship Calendar

Subscribe to the unofficial calendar

The data in this calendar is coming directly from www.onefc.com/events/. There are no ads, data collection, or trackers embeded in the events. This is 100% free!

You can subscribe to the unofficial fight calendar using your method of choice.

Run your own server

The docker image is also available for anyone to download and run on their own server. Instructions are available on the gitlab wiki.

This is a free and open sourced project created by Richard Soper for fun! Source code is available on my gitlab instance. This project is not affiliated to ONE Championship directly.
