What's the catch?

The data in these calendars are coming directly from the events page for the MMA organization linked above. There are no ads, data collection, or trackers embeded in the events. This is 100% free and it always will be.

Run your own server

The docker image is also available for anyone to download and run on their own server. Instructions are available in this post.
